Password Generator

Based on a random number algorithm I learned in Python, this application was created to produce passwords that are secure and effective. I used Ubuntu Bash to test the algorithm. I used a CSRF token to create a form resubmission notice that pops up when the page is refreshed. I chose to use Bootstrap for my front end for practice since I was learning it. I originally deployed the application to AWS using Ubuntu Bash shell scripting.

Mars Attacks

This is the first game I built using front-end technologies HTML, CSS, and JavaScript. Each card responds to a click by flipping to reveal a symbol on the other side. Only two cards can be chosen at a time. If the cards match, they remain on the screen, if they do not, they flip back to their original position. When the user wins, a message pops up.
